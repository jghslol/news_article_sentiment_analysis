SENTIMENT ANALYSIS OF TODAY'S NEWS: THE GUARDIAN, THE MAIL AND THE METRO


This is a sentiment analysis of today's news of three major news sources: The Guardian, The Metro and the Daily Mail. The tool uses python web scraping library beautiful soup to scrape the article and title data for three of the top three stories from today's news on each site if there are three or less if there are less than three. If there are no articles at all, or you do not select any box, no data will be returned. It uses the natural language processing toolkit in python to do the sentiment analysis and to provide an article summary and a summary of the title. The tool is also written in PHP and HTML and it uses the Dompdf library to export the results to a PDF file for the user.

Please check the boxes for the news sources that you would like analysis on and click submit and wait a few seconds to get a returned PDF document of the results.
